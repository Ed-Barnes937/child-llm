# Age Tiers — Questions to Discuss

- What's the minimum age we'd target? Is 4 too young for an AI chat experience?
- What's the upper age? 15? 16? At what point do they just use a normal LLM?
- Should we use fixed tiers (e.g. 4-6, 7-9, 10-12, 13-15) or just ask for the child's birthday and apply a sliding scale of defaults?
- If fixed tiers, how many? 3 feels too coarse, 5 might be overkill.
- Should tiers be labelled by age, school stage, or something friendlier (e.g. "Explorer", "Discoverer")?
- Should parents be able to override the tier defaults — e.g. move a mature 7-year-old up, or a less confident 11-year-old down?
- At what age should kids move from structured prompts ("I want to learn about...") to freeform typing?
- At what age should the AI start giving more direct answers vs. guiding with questions?
- Should younger children have shorter sessions or message limits?
- At what age do parents stop seeing full conversations and start seeing only summaries or flagged items?
- Is there an age where parental review should be opt-in rather than default?
